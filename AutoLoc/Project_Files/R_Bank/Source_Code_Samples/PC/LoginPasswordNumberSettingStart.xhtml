セキュリティボード
暗証番号は、ログインパスワードとは異なり、決済を行う際に本人確認のためにご入力いただくものです。

暗証番号の条件

・半角数字のみ4桁~12桁
・記号（"-"や"_"）は利用できません。
・他人から推測されやすい文字列（生年月日、電話番号、同一の数字など）は暗証番号に設定できません。
キャッシュカードを申込済、または今後申込予定の方へ
ATMで残高照会やお引き出しを行う際に使う キャッシュカードの暗証番号 は、 ここで設定した暗証番号の頭4桁 を利用します。
【例】 ログイン後のお取引で利用する暗証番号が「 12345678 」の場合、キャッシュカード暗証番号は「 1234 」です。
