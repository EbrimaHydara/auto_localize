楽天銀行アプリ認証の初期設定を行う

楽天銀行アプリ認証の初期設定を行います。
ご利用には、「楽天銀行アプリ for Business」のダウンロードが必要です。
Step 1
セキュリティ担当者 → 担当者（ユーザ）
楽天銀行アプリ認証を利用する担当者（ユーザ）あてに、初期設定に必要な認証コードを発行します。
Step 2
担当者（ユーザ）
「楽天銀行アプリ for Business」をダウンロードして、ログインします。
Step 3
担当者（ユーザ）
PCサイトで認証コードを確認し、「楽天銀行アプリ for Business」の初期設定画面で、認証コードを入力します。
「リモートバンキング利用規定」を確認する
規定に同意して、楽天銀行アプリ認証の初期設定を行います。
