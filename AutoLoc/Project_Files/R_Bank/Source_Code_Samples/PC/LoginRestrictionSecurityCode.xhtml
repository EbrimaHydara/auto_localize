ログイン制限サービス

ここにエラーメッセージが表示されます
ログイン制限サービスには、２つの使い方があります。
・ログインに必要なワンタイムキーを電子メールで通知します。
・ログインの都度、携帯電話でログイン制限を一時解除して利用できます。
通常の「ログインパスワード」以外に二重の本人認証をすることでお客さまの口座をガードします。ログインをする際に、電子メールでお届けする「ワンタイムキー」を入力してロックを解除する方法と、登録済みの携帯電話を鍵代わりにしてロックを解除する方法の両方を利用できます。
