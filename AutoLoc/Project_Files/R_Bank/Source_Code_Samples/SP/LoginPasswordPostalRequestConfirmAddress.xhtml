お届け先住所を確認します。
変更がなければ、チェックボックスにチェックを入れて、ログインパスワード設定用の番号の郵送発行を申し込んでください。

※セキュリティ強化のため、お届先住所の下5文字のみを表示しています。

下5文字の表示でわからない場合、または住所変更があった場合は、手続きを中止して、楽天銀行カスタマーセンターまでご連絡ください。

楽天銀行カスタマーセンター 0120-330-507
（受付：年中無休　24時間）

お手続きを中止する
  • ログインパスワード設定用番号は、「転送不要」郵便で翌営業日以降順次発送いたします。
    「転送届」を出されている場合、お届けすることができません。
  • 現在設定されているログインパスワードは無効となり、発行された最新のログインパスワード設定用番号が有効となります。
