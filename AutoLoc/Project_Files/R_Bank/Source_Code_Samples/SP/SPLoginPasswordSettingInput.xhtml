ログインパスワードを新たに設定してください。

ご自身で設定いただいたスピード開設初回ログイン用パスワードと同様のものを再度設定することもできます。

ログインパスワードの条件

以下①～④の4種類をすべて使用し、8～12桁（すべて半角）で設定してください。

  • ①英大文字
  • ②英小文字
  • ③数字
  • ④記号  
    • 使用できる記号は $ - . / : @ [ ] _ # & ? です。
  • 8桁～12桁（すべて半角）
入力されたパスワードが一致しません
