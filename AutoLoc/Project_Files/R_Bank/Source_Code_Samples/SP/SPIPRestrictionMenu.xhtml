PCサイトで設定したIPアドレス制限を一時解除します。
一時解除すると、30分間、登録していないIPアドレスやドメインからも取引・手続が可能になります。

IPアドレス制限について

普段利用するIPアドレスやドメインネームを事前に登録することで、登録以外のIPアドレスからアクセス（ログイン）している場合の取引を制限するサービスです。
設定はPCのみで行うことができます。

IPアドレス制限の詳細はこちら
現在の設定
IPアドレス制限が設定されています。　　　　

設定を完了するには、次の画面で認証を行う必要があります。

  • IPアドレス制限の設定はPCサイトから行うことができます。
  • パソコン用に設定中のIPアドレス制限を一時解除するものであり、モバイル端末ログイン制限は一時解除されません。
