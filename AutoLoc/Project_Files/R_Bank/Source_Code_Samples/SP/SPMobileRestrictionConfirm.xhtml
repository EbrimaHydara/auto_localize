モバイル端末ログイン制限について

スマートフォン端末からのログイン方法を制限することで、認証強化をすることができます。
モバイル端末を利用するお客さまは、「クイックログインによるアクセスのみ許可する」を推奨します。

端末の機種変更、あるいはスマートフォン向けアプリケーションの再インストールを行う場合は、ご変更前に必ず「全てのアクセスを許可する」に設定をしてください。それ以外の設定では、新しい端末からログインできなくなる場合があります。

    　
  • 「スマートフォン向けアプリケーション」とは、iPhoneおよびAndroid向けに提供しているアプリケーションです。
    モバイル端末ログイン設定は、スマートフォン向けアプリケーションでも適用されます。
