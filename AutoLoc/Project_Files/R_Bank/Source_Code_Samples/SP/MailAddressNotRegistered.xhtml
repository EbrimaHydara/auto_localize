セキュリティ

ご注意ください

メールアドレスを登録してください。

楽天銀行自慢の豊富なセキュリティ機能（手数料無料）、取引通知メール、かんたん振込（メルマネ）をご利用いただくには、まずメールアドレスの登録が必要です。
セキュリティ機能だけではなく、定期預金の解約などにも使いますので、必ず登録をお願いいたします。
既にメールアドレスを仮登録済みの場合は「仮登録情報」よりお進みください。
